# FridgeOS

Triển khai lại theo đúng hướng dẫn công nghệ và cấu trúc trong PRD:

- React 18 + Vite + TypeScript
- Zustand store
- Domain/Application/Infrastructure/UI layers
- Route theo modules: `/fridge`, `/recipes`, `/suggest`
- PWA plugin scaffold (`vite-plugin-pwa`)

## Run

```bash
npm install
npm run dev
```
